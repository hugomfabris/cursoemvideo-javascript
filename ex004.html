Importante lembrar que foram vistos métodos no Node.js:

Alternativa para concatenação, o template String ´${}´

Template Strings são strings que permitem expressões embutidas. Você pode utilizar string multi-linhas e interpolação de string com elas.

Método para fixar casas decimais: .Tofixed

Método para converter moedas: n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}
