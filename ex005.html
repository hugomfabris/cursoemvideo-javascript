Iniciando estudo de DOM

Árvore DOM, discussões

A estrutura se comporta como um fluxograma, no qual temos os ojetos "parents" dando origem aos objetos "child".

No exemplo da aula o objeto window origina location, history, document. Este último por sua vez origina html, que origina vários outros. As tags dentro de tags serão consideradas os objetos que são os componentes da árvore dom. Podemos navegar de diversas maneiras pelos elementos das árvoes.

As seleções podem ser feitas, notavelmente, por: Marca, ID, Classo, Nome e Seletor. Os seletores ficam dentro das tags (no exemplo, da div) e as funções usadas são: getElementsBy___[] ou getElementBy().

Lembrete de aulas anteriores: o comando ctrl shift P serve para envelopar o texto, ganhando agilidade na digitação.
